# Mini Shop

小型购物网页平台 — 用户注册登录、商品浏览购物车下单、管理员商品增删改查 + 订单管理。

## 技术栈

- **前端**: HTML / CSS / JavaScript (原生)
- **后端**: Node.js + Express
- **认证**: JWT + bcryptjs
- **数据存储**: JSON 文件
